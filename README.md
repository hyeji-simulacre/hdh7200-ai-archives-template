## Quick Start — How to Import This Workspace / كيفية استيراد مساحة العمل

```bash
git clone https://github.com/[INSTRUCTOR_REPO]/AI-Archives-Workspace-HDH7200.git
```

Open the folder in **Antigravity** → File → Open Folder.

### Option B — Download ZIP (no Git account needed)

1. Click the green **Code** button on this GitHub page

## Five Rules / خمس قواعد

1. **Read each folder's README before using it** / اقرأ ملف README لكل مجلد قبل استخدامه
2. **Always record the source and license** when collecting / سجّل المصدر والترخيص دائمًا عند الجمع
3. **Never delete originals** — keep the original even after conversion / لا تحذف الأصول أبدًا
4. **Review agent output before citing it** — AI can hallucinate / راجع مخرجات الوكيل قبل الاستشهاد بها
5. **You are the author** — even when an agent helps you write / أنت المؤلف — حتى عندما يساعدك الوكيل
