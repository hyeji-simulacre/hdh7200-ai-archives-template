# 50-daily-report — Daily Activity Log / سجل النشاط اليومي

**EN** — **One file per day.** What did you collect today? What did you create? What did you learn? What will you do tomorrow? This is not a personal diary — it is your **archive activity log**.

**AR** — **ملف واحد لكل يوم.** ماذا جمعت اليوم؟ ماذا أنتجت؟ ماذا تعلمت؟ ماذا ستفعل غدًا؟ هذا ليس يوميات شخصية — بل هو **سجل نشاطك الأرشيفي**.

---

## Why Daily Notes / لماذا الملاحظات اليومية

An archive is not built in a day. It is built **one day at a time**. The daily note is how you keep going:

- You remember what you did last week
- You notice patterns in how you collect
- You catch problems early
- You have something to show when someone asks "what did you work on?"

> Aim for **3–5 daily notes per week** during practice weeks (Week 5–10).
