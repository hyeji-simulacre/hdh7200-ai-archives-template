# 51-daily-note — Daily Notes / الملاحظات اليومية

**EN** — One Markdown file per day. File name: `YYYY-MM-DD.md`. Use `@daily-note` at the end of each session to auto-generate.

**AR** — ملف Markdown واحد لكل يوم. اسم الملف: `YYYY-MM-DD.md`. استخدم `@daily-note` في نهاية كل جلسة للتوليد التلقائي.

---

## Template / القالب

```markdown
# YYYY-MM-DD

## What I collected today
- [list of records added to 30-collected/]

## What I created today
- [list of notes added to 40-created/]

## What I learned
- [observations, problems, discoveries]

## Tomorrow
- [what you plan to do next]
```
